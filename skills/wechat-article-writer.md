# 微信公众号文章创作

## 能力
- 根据用户需求创作微信公众号文章
- 支持多种文章类型（工具推荐、技术科普、行业洞察、生活分享）
- 自动生成 Markdown 格式内容，结构清晰
- 支持插入网页截图和图片

## 文章风格
- 标题醒目，能吸引读者
- 段落分明，每段 2-4 句话
- 适当使用列表、引用等格式
- 结尾有总结或引导互动

## 图片插入机制
文章中使用特殊标记，后续处理时替换为实际图片：

- `[IMAGE:url_or_description]` - 普通图片，需要用户提供 URL 或描述
- `[SCREENSHOT:url]` - 网页截图，系统会自动调用截图脚本

示例：
```
[SCREENSHOT:https://tools.aiyax.com]
[IMAGE:一张展示 AI 工具界面的截图]
```

## 输出格式
直接输出 Markdown 格式的文章内容，包括：
1. 文章标题（二级标题）
2. 正文内容（带适当格式）
3. 图片标记（如需要）

## 注意事项
- 不输出代码块包裹的 Markdown，直接输出内容
- 图片标记只用于需要配图的地方
- 文章长度适中，一般 800-2000 字
- 保持客观、专业的语气
